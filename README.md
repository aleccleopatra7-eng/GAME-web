Run locally:
1. (Optional) npm install
2. Serve the folder with a simple static server (live-server, http-server, or VS Code Live Server).
3. Open http://localhost:8080 (or the port your server uses).
Controls:
- Click canvas to lock pointer and start playing.
- WASD to move, Mouse to look, Space to jump.
Objective:
- Collect golden spheres to gain score and small healing.
- Avoid or outrun red enemies; if they touch you you lose health.
This is a minimal baseline. Extend by adding levels, models, sound, UI, and more sophisticated physics.
